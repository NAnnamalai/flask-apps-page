Simple CRUD Operation with Restaurant and it's Menu Items

    What are the features ?
  • list all the restaurants, clicking on a restaurant will show all the menu items under it
  • will be able to update restaurant and menu items details with edit links beside them
  • will be able to delete restaurant and menu items individually
    Routes
  • / and /list_restaurants displays all the restaurant names
  • /view_menu gets triggered when clicking on a restaurant name
  • new restaurants are added using /new_res , where details are submitted to a POST request to /add_res
  • new restaurants are added using /new_men , where details are submitted to a POST request to /add_men
  • /edit_res gets trigered when clicking on edit link, which inturn submits a POST request to /update_res
  • /delete_res will delete the restaurant
  • /edit_men gets trigered when clicking on edit link, which inturn submits a POST request to /update_men
  • /delete_men will delete the menu item
